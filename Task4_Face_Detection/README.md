# Task 4 - Face Detection

This project is part of the CODSOFT Artificial Intelligence Internship.

## Project Description
This Python program detects faces in real-time using a webcam.

## Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## How to Run

1. Install OpenCV
pip install opencv-python

2. Run the program
python face_detection.py

3. Webcam will start and detect faces.

Press ESC to exit the webcam window.

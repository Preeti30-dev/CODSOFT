## ðŸ§  How the AI Works

The Minimax algorithm works by:

- Simulating all possible future moves
- Assuming the opponent plays optimally
- Assigning scores to outcomes (win = 1, loss = -1, draw = 0)
- Choosing the move with the best possible score

This ensures the AI never loses.
